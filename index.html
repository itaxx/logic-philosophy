Logic for Philosophy (exercises).

Working through Ted Sider's Logic for Philosophy with some other philosophers here at UCL, I decided that it would be nice to experiment using GitHub to compile a document with the answers to the exercises in the book.

At this stage, it is possible to access the TeX file, comment on it and suggest changes. I am a novice user of Git, and so will have to see if this is actually an effective strategy for collaboration. If anyone of the group is interested in experimenting with more of Git's functionality, let me know.

For those that just wish to download the most recent version of the PDF file, they can find it here.
